🐱 About Party Cat

🎉 What is Party Cat?

Party Cat is a free, open-source party planning app, built by me, Maeve Lander.

I am not, in fact, a party animal myself. But I do love a good list to help ward off party cat-astrophies! I built this kitten-kaboodle to make organising my kids' birthday parties fun and easy. I hope Party Cat makes you smile, and helps make your own party plans purrrfect.

🔧 Technical Details

Built with: HTML, CSS, and JavaScript—no complex frameworks, just clean, efficient code

Storage: Local browser storage for guest mode, Supabase for cloud accounts

Privacy: No tracking, no analytics (beyond basic anonymous usage stats), no data selling

Open Source: The code is available for anyone to view, modify, or contribute to
